# DineHub (Logic Prototype)

DineHub is a mobile application that helps users discover restaurants, filter them by budget, view details, and reserve tables with manual payment confirmation (receipt upload).

## Features
- User authentication (prototype)
- Restaurant discovery & listing
- Search and budget-based filtering
- Restaurant details screen
- Table reservation flow
- Manual payment proof upload (File Upload/Download)
- Location-based discovery (GPS)
- Push notifications (conceptual)

## Screens (Figma + App)
- Login Screen
- Home / Restaurant List Screen
- Restaurant Details Screen
- Reservation Screen
- Upload Payment Proof Screen

## How to Run
This repository is a logic prototype showing planned architecture, screens, services, and data models.
The clickable UI prototype is available in Figma.
